# 如何运行Frontend程序

## 方法1: 直接使用系统Python（推荐，不需要虚拟环境）

### 步骤1: 安装依赖

```powershell
# 导航到Frontend文件夹
cd D:\Pycharm_Data\PythonProject\Frontend

# 安装所有依赖（直接安装到系统Python）
pip install -r requirements.txt
```

### 步骤2: 运行程序

```powershell
# 启动Streamlit应用
streamlit run app.py --server.maxUploadSize=2048

# 或者使用Python模块方式
python -m streamlit run app.py --server.maxUploadSize=2048
```

### 步骤3: 访问应用

浏览器会自动打开，或手动访问：http://localhost:8501

---

## 方法2: 使用虚拟环境（可选）

如果你想要使用虚拟环境（隔离依赖）：

### 创建虚拟环境

```powershell
cd D:\Pycharm_Data\PythonProject
python -m venv .venv
```

### 激活虚拟环境

```powershell
# PowerShell
.\.venv\Scripts\Activate.ps1

# 或CMD
.venv\Scripts\activate.bat
```

### 安装依赖并运行

```powershell
cd Frontend
pip install -r requirements.txt
streamlit run app.py --server.maxUploadSize=2048
```

## 常见问题

### Q: 提示"streamlit不是内部或外部命令"

**解决**：使用 `python -m streamlit run app.py` 代替 `streamlit run app.py`

### Q: 提示"No module named 'streamlit'"

**解决**：先安装依赖 `pip install -r requirements.txt`

### Q: 端口被占用

**解决**：使用其他端口 `streamlit run app.py --server.port 8502 --server.maxUploadSize=2048`
