# 🚀 快速启动指南

## 方法一：使用命令行启动（推荐）

### 步骤1：打开终端/命令提示符

**Windows系统：**
- 按 `Win + R`，输入 `cmd` 或 `powershell`，按回车
- 或者右键点击"开始"按钮，选择"Windows PowerShell"或"命令提示符"

### 步骤2：导航到Frontend文件夹

```powershell
# 假设项目在 D:\Pycharm_Data\PythonProject
cd D:\Pycharm_Data\PythonProject\Frontend
```

### 步骤3：检查并安装依赖

```powershell
# 检查是否已安装streamlit
pip show streamlit

# 如果未安装，安装所有依赖
pip install -r requirements.txt
```

### 步骤4：启动应用

```powershell
# 启动Streamlit应用（支持最大2GB文件上传）
streamlit run app.py --server.maxUploadSize=2048
```

### 步骤5：打开浏览器

启动成功后，终端会显示类似以下信息：
```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://192.168.x.x:8501
```

**自动打开：** Streamlit通常会自动在默认浏览器中打开应用。

**手动打开：** 如果没有自动打开，复制 `http://localhost:8501` 到浏览器地址栏访问。

## 方法二：使用批处理文件启动（Windows）

### 创建启动脚本

在 `Frontend` 文件夹中创建一个名为 `启动.bat` 的文件，内容如下：

```batch
@echo off
echo 正在启动风机叶片视频预处理系统...
cd /d %~dp0
streamlit run app.py --server.maxUploadSize=2048
pause
```

**使用方法：** 双击 `启动.bat` 文件即可启动应用。

## 方法三：在PyCharm中启动

### 步骤1：在PyCharm中打开项目

1. 打开PyCharm
2. 打开项目文件夹：`D:\Pycharm_Data\PythonProject`

### 步骤2：配置运行配置

1. 点击右上角的运行配置下拉菜单
2. 选择 "Edit Configurations..."
3. 点击 "+" 添加新配置
4. 选择 "Python"
5. 配置如下：
   - **Name**: Streamlit App
   - **Script path**: `D:\Pycharm_Data\PythonProject\Frontend\app.py`
   - **Parameters**: `run app.py --server.maxUploadSize=2048`
   - **Working directory**: `D:\Pycharm_Data\PythonProject\Frontend`
   - **Python interpreter**: 选择你的Python解释器

### 步骤3：运行

点击运行按钮，PyCharm会在内置终端中启动应用。

## 常见问题解决

### 问题1：提示 "streamlit不是内部或外部命令"

**解决方法：**
```powershell
# 使用python -m方式启动
python -m streamlit run app.py --server.maxUploadSize=2048
```

### 问题2：端口8501已被占用

**解决方法：**
```powershell
# 使用其他端口启动
streamlit run app.py --server.port 8502 --server.maxUploadSize=2048
```

然后在浏览器中访问：`http://localhost:8502`

### 问题3：依赖安装失败

**解决方法：**
```powershell
# 升级pip
python -m pip install --upgrade pip

# 重新安装依赖
pip install -r requirements.txt
```

### 问题4：浏览器没有自动打开

**解决方法：**
1. 查看终端中显示的URL（通常是 `http://localhost:8501`）
2. 手动复制URL到浏览器地址栏
3. 或者点击终端中显示的链接（如果支持）

## 停止应用

在运行应用的终端窗口中，按 `Ctrl + C` 即可停止应用。

---

## 访问地址说明

- **本地访问**: `http://localhost:8501` （只能在本机访问）
- **局域网访问**: `http://你的IP地址:8501` （同一局域网内的其他设备可以访问）
